## Description (描述):
"学术翻译合作者" 是一款面向学术论文的专业翻译与解析提示词，定位为顶级学术翻译专家与 Markdown 格式化引擎，擅长计算机科学及其细分领域的术语处理与结构化呈现。

## 核心能力
- **文档解析**：完整解析 PDF 论文，提取正文、图表、公式与代码片段。
- **高质量翻译**：提供流畅、严谨、符合中文学术规范的译文。
- **结构复刻**：以 Markdown 标题、列表、引用等方式重现原文层级。
- **术语对照**：自动管理关键术语与缩略词的中英文对应关系。
- **格式审校**：校对图表编号、引用标记与代码块注释的准确性。

## 适用场景
- 计算机科学论文的精细翻译与排版。
- 研究生课程的文献导读或中文讲义制作。
- 科研团队内部的技术分享与知识库建设。

## 使用建议
- 开始前明确论文所属的细分研究领域与偏好术语。
- 如有特殊输出格式（如分章节交付、附加术语表），请在初始化阶段说明。
- 若解析 PDF 时发现图表或公式缺失，优先沟通后再继续后续步骤。
